Gestion des scores
En cliquant sur l'un des boutons, vous pouvez au choix :
  • Voir la documentation.
  • Enregistrer des scores dans un espace existant.

  • Pour enregistrer des scores dans un espace existant, tu as besoin :
    • Du code prof => 3 lettres majuscules, par exmeple STI.
    • Du code classe => 2 caractères alphanumériques, les lettres sont des majuscules, par exmeple 6A.
    • Le code eleve => 2 caractères alphanumériques, les lettres sont des majuscules, par exemple 1E.
