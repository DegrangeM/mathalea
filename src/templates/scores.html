Profs - Espace de gestion des scores


Vous pouvez au choix :
  • Créer un espace de sauvegarde pour vos classes, vous obtenez :
    • Un code prof public de trois lettres majuscules, par exemple STI
    • Un lien vers l'espace web où seront stocker les résultats de vos élèves :
      • Un fichier csv par semaine et par classe avec un nom générique, par exemple semaine27.csv si on est la 27e semaine de l'année.
      • Par exemple, pour la classe de 6A, le fichier contiendra tous les résultats de la semaine 27 pour tous les élèves de la classe de 6A.
  • Pour pouvoir enregistrer des scores dans un espace existant l'élève doit se connecter avec un userId au format suivant :
    • Le code prof => 3 lettres majuscules, par exemple STI.
    • Le code classe => 2 caractères alphanumériques, les lettres sont des majuscules, par exemple 6A.
    • Le code eleve => 2 caractères alphanumériques, les lettres sont des majuscules, par exemple 1E.
    Le code élève est alors STI6A1E, une fois connecter ce code apparait dans la banière de titre des exercices.
